Production-ready token deployment and distribution workflows.

Token Distribution bundles guided Node.js wizards for ERC-20 deployment and transfers, helping operators launch tokens, send supply safely, and keep a traceable record of each run.

Quick start View repository

Deploy once

Create an ERC-20 token and mint the full initial supply directly to the recipient wallet in a single guided flow.

Transfer confidently

Use the transfer wizard to send existing token balances with validation, balance checks, and receipt capture.

Keep an audit trail

Each execution writes structured JSON reports so distribution activity can be reviewed, shared, and archived.

Why teams use it

A lightweight operator console for live token operations.

  • Guided prompts for network, wallet, token metadata, and recipients
  • Supports dry runs before broadcasting any on-chain transaction
  • Works with environment-based configuration or interactive input
  • Built on Node.js and ethers for familiar operational tooling

Core commands

Install dependencies npm ci
Launch deployment wizard npm start
Run without broadcasting npm run dry-run
Launch transfer wizard npm run transfer

Operational flow

1. Configure

Set a private key and RPC URL in a local .env file, or provide them interactively at runtime.

2. Validate

Use dry-run mode and the built-in prompts to confirm token metadata, supply, and recipient destinations.

3. Execute

Broadcast the transaction only after review, then keep the generated report as part of your operations record.

Production notes
